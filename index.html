JavaScript Practice App

I wrote this app as a way to create a tool for myself to practice working with JavaScript date constructors. The app generates a small snippet of JavaScript and places it in a psuedo code editor. You can answer the questions by coding in the correct answer and then submitting your response. Currently, the questions revolve exclusively around what values will be returned when local methods versus utc methods are called on Date object that's generated. I wrote it in vanilla JS to practice fundamentals. I'd like to write a more full-fleged app in a framework that would give me and others a way to practice JavaScript fundamentals that feels like you're coding instead of only creating and then practicing static flashcards.

thebeltbook.com

The beltbook is a awesome web app to help Musical Theatre performs manage their repertoire. I connected to the project through a voice teacher that I had when I was growing up. It's written in React and Python/Django with a MySQL database. I didn't write the app itself. I was approached by the owner after the developer who wrote most of it was no longer available to work on the project. I was tasked with getting it to production. I provisioned a Digital Ocean droplet running Ubuntu and I've deployed the app using Caddy and Gunicorn. Alongside deployment I've connected the app to use a 3rd-party email provider, implemented cache busting, and configured the DNS setup.
